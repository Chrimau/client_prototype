    Second Semester Project

    Deploying a web page and provisioning a web server


    About This Project

    This project showcases a foundational step in web application development.

    As part of our task, we provisioned a Linux server in the cloud and set it up to host a fully functional landing page.
    This involved installing and configuring a web server, deploying custom HTML, CSS, and JavaScript files, and fine-tuning the server to handle incoming HTTP traffic.


    The landing page serves as a demonstration of technical capabilities, featuring a sleek design and interactivity, all hosted on a robust cloud infrastructure.

    This exercise highlights core competencies in server provisioning, web deployment, and system optimization, reflecting the skills and dedication required to build scalable and impactful digital solutions.
